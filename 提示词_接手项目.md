# 接手项目提示词

请按接力开发规则接手当前目录，并严格执行以下顺序：

1. 读取 `AGENTS.md`
2. 读取 `task_registry.md`
3. 读取 `progress.md`
4. 读取 `task_plan.md`
5. 读取 `findings.md`
6. 读取 `避坑记录.md`
7. 读取 `修改记录_会话备忘.md`

输出以下内容：
- Task-ID / Task-Name
- Active Goal
- Current Phase
- Next First Command
- Open TODO Top3
- Known Avoidances Top3

然后执行 `Next First Command`，再继续当前任务。
