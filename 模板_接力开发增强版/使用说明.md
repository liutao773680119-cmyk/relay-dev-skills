# 使用说明

## 最快用法
在目标项目目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\模板_接力开发增强版\init-project.ps1
```

如果模板包不在目标目录，先把整个 `模板_接力开发增强版` 文件夹复制过去，再执行脚本。

## 可选参数
```powershell
powershell -ExecutionPolicy Bypass -File .\模板_接力开发增强版\init-project.ps1 -TargetDir . -TaskId T001 -TaskName "初始化项目接手"
```

## 初始化后要做的事
1. 运行 `Get-ChildItem -Force`
2. 如果有 git，运行 `git status --short`
3. 用 `提示词_接手项目.md` 开始新会话
4. 后续一旦确认失败路径，更新 `避坑记录.md`
